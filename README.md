Computer Vision Lab – Image Acquisition & Filtering (MATLAB)

This project implements fundamental computer vision concepts using MATLAB, focusing on image acquisition, camera parameter analysis, noise characterization, and image filtering techniques.

Features
Image acquisition using MATLAB Image Acquisition Toolbox (IAT)
Analysis of camera parameters (exposure, gain, focus, aperture)
Evaluation of image quality (contrast, sharpness, histogram analysis)
Camera characterization (resolution, sensor size, pixel size, field of view, depth of field)
Noise measurement and statistical analysis
Implementation of denoising filters (binomial, median)
Edge detection using Sobel, Prewitt, Laplacian, and MATLAB edge() function
Comparative analysis of filtering techniques

Modules
Image Acquisition – Capturing and analyzing images via IAT
Camera Analysis – Studying physical and optical camera properties
Noise Analysis – Measuring and modeling noise behavior
Filtering – Applying smoothing and denoising techniques
Edge Detection – Extracting image features using gradient-based operators

How it works
Images are captured and analyzed under varying camera and lens settings to study their impact on image quality. Noise is measured under controlled conditions and reduced using filtering techniques. Finally, edge detection algorithms are applied to evaluate feature extraction performance.

Tools & Technologies
MATLAB
Image Acquisition Toolbox (IAT)

This lab provides hands-on experience with the complete imaging pipeline, from data acquisition to image enhancement and feature extraction.
