# Computer Vision: Image Acquisition & Filtering (MATLAB)

This project explores core computer vision concepts using MATLAB, focusing on image acquisition, camera parameter analysis, noise characterization, and image filtering techniques.

## Features

* Image acquisition using MATLAB Image Acquisition Toolbox (IAT)
* Analysis of camera parameters (exposure, gain, focus, aperture)
* Image quality evaluation (contrast, sharpness, histogram analysis)
* Camera characterization (resolution, sensor size, pixel size, field of view, depth of field)
* Noise measurement and statistical analysis
* Denoising using binomial and median filters
* Edge detection using Sobel, Prewitt, Laplacian, and MATLAB `edge()` function
* Comparative analysis of filtering techniques

## Modules

* **Image Acquisition** – Capture and analyze images using IAT
* **Camera Analysis** – Evaluate optical and sensor properties
* **Noise Analysis** – Measure and model noise behavior
* **Filtering** – Apply smoothing and denoising techniques
* **Edge Detection** – Extract features using gradient-based operators

## How it Works

Images are captured under varying camera and lens settings to analyze their impact on quality. Noise is measured under controlled conditions and reduced using filtering techniques. Edge detection methods are applied to evaluate feature extraction performance.

## Tools & Technologies

* MATLAB
* Image Acquisition Toolbox (IAT)

This project demonstrates the complete imaging pipeline, from data acquisition to enhancement and feature extraction.
